Actividad de cierre 43. Marca o colorea partes para encontrar el área.

El área de este rectángulo se puede encontrar hallando \(6 \times 7\text{.}\)
  1. Marca o colorea el rectángulo para mostrar que podemos escribir \(2 \times (3 \times 7)\) o \((6 \times 5) + (6 \times 2)\) para encontrar su área.
  2. ¿Cuál es el valor de \(6 \times 7\text{?}\) Explica o muestra tu razonamiento.
